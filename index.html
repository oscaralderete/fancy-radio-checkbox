Fancy radio & checkbox inputs

This CSS is based on https://codepen.io/aaroniker/pen/ZEYoxEY, I was adapting that on a daily work project when, suddendly, after some erratic modifications... whoa, that muted on this!

I'm a developer not a CSS guy, I'm sharing it because I think it could be useful to others, so if you have some complains about "semantic" or how I set CSS rules or whatever, now you know why...

IMPORTANT! There is a quirky behaviour on narrow screens and multiple option columns (as happens in this current page layout), the background image desplaces in an ugly way, this is because the CSS rules operate on <label> tag. I guess you can fix it wraping the <input> tag with a <span> and then applying the background image on the <span> tag, but it seems to overload the markup. Furthermore in my personal experience, most cases of forms containing options/checkboxes use a single column layout.

You can find a running version on https://experiments.oscaralderete.com/fancy-radio-checkbox/

PS: To show serialized data I'm using this cool plugin: SerializeJSON jQuery plugin.
